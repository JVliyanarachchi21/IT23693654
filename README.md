# IT23693654 Test Automation Assignment

## Setup

Install dependencies:

```
pip install playwright openpyxl
```

Install Playwright browsers:

```
python -m playwright install
```

## Running Tests

Run the script using:

```
python test_automation.py --excel "IT23693654.xlsx" --url "https://www.pixelssuite.com/chat-translator"
```
Note: Most test cases are negative test cases designed to produce FAIL results.
